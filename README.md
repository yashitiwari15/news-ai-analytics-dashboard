# AI News Intelligence Dashboard

An AI-powered analytics platform that collects news data, performs sentiment analysis, topic detection, and visualizes insights using an interactive dashboard.

Features
- Automated news data pipeline
- Sentiment analysis
- BERT topic clustering
- LLM-powered news summarization
- 3D article analytics
- Interactive Streamlit dashboard

Tech Stack
Python, Streamlit, Plotly, HuggingFace Transformers, Sentence Transformers, SQLite
